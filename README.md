# Stock Market ELT Pipeline

A production-style batch ELT pipeline that fetches daily stock market data and loads it into PostgreSQL for analysis.

## Tech Stack
- **Python** — extract and load logic
- **Apache Airflow** — orchestration and scheduling
- **PostgreSQL** — data warehouse
- **Docker** — containerized setup
- **Power BI** — dashboard and visualization
- **Alpha Vantage API** — free stock market data source
